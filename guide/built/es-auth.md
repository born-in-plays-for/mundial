# No Connection to the Server
<!-- /i18n:offline_page_title -->

<!-- i18n:offline_intro -->
The little person icon in the top-right corner is also where you'll see it if the app can't reach its backend server.
<!-- /i18n:offline_intro -->

<!-- i18n:offline_body -->
## What "offline" means here

Signing in, the live game page, and remembering your session all depend on a small backend server. That server runs on the developer's own machine and isn't always switched on — when it's off (or unreachable), the navbar shows a warning icon and a WhatsApp icon instead of the normal sign-in button, on a plain beige background.

Everything else keeps working normally: the map, the country rankings, the player list, and every filter/sort/keyboard shortcut described in the other guide — none of that needs the backend at all.

## Getting help

Tap the warning icon to open a short explanation with a WhatsApp link — message that number and the server can usually be turned back on within a few minutes.

## It fixes itself

No need to reload the page: the app quietly retries the connection every 30 seconds in the background, and the sign-in button reappears on its own as soon as the server answers again.
